#SteamSYWC使用手册<br>
##项目描述<br>
实现一个命令行文本计数统计程序。能正确统计导入的纯英文txt文本中的字符数，单词数，句子数。<br>
##使用方法<br>
 1. 打开exe文件：路径为WC\\DEBUG\\WC.exe<br>
 2. 输入命令行：命令行有效指令，如下<br>
  （1）计数字符数：wc.exe-c file.txt<br>
  （2）计数单词数：wc.exe-w file.txt<br>
  （3）计数句子数：wc.exe-s file.txt<br>
 3. 程序返回计数值<br>
